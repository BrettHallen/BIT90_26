# BIT90 Original Schematics
Recreated from the [service manual schematics](https://github.com/b3rendsh/bit90/tree/main/docs) with errors corrected, or questioning comments when I couldn't figure it out.<br>

Planning to use these as a basis for a recreated machine - these are not meant to be used for a one-to-one recreation (i.e. not bothering assigning footprints or working on the PCB file) ... but do what you wish with them.<br>

Any errors are either slavishly copied from the original or misinterpreted by me.<br>
